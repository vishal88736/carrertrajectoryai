## 📐 Future Potential Score (FPS)

```
FPS = 0.35 × Semantic Fit
    + 0.30 × Career Momentum
    + 0.20 × Behavioral Evidence
    + 0.15 × Contextual Intelligence

Career Momentum = √(Career Velocity × Direction Alignment)
```
